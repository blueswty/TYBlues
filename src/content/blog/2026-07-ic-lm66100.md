所属大类是功率电子开关。很多人之前可能用过[**SY6280AAC**](https://item.szlcsc.com/56162.html?fromZone=s_s__%22%E5%8A%9F%E7%8E%87%E5%BC%80%E5%85%B3%22&spm=sc.gbn.xh2.zy.n___sc.hm.hd.ss&lcsc_vid=EVlZAQEAFgdXV1BXQVNWUVZeRFNcBgBQQwNeBFJXE1ExVlNeQ1lZUlRSRFRZVztW)、[**MT9700-N**](https://item.szlcsc.com/44491709.html?fromZone=s_s__%22%E5%8A%9F%E7%8E%87%E5%BC%80%E5%85%B3%22&spm=sc.gbn.xh5.zy.n___sc.hm.hd.ss&lcsc_vid=EVlZAQEAFgdXV1BXQVNWUVZeRFNcBgBQQwNeBFJXE1ExVlNeQ1lZUlRSRFRZVztW) 这种常规的单向开关，在 USB 接口处起到限流及单向导通作用。

不得不说，SY6280 使用相当频繁，在含 USB 接口的消费产品中随处可见，它具备的单向导通 + 主动限流功能非常实用可靠。

![image-20260724145117067](https://picgocloud.com/m/cc1fc314-a59a-4f7f-ae2f-07c43664eb78.png)



这次介绍的 LM66100 同样具备单向导通能力，但是和 SY6280 的差异在于：

LM66100 可以应用于多电源供电的无缝选通电路中，可以承担更复杂电源架构下的解决方案。

![image-20260724145322284](https://picgocloud.com/m/2d7e3f3b-b342-4f33-bb49-0169402a754a.png)

## 使用介绍

### 关键功率参数

1. 电压范围 `1.5V~5.5V`；优于 SY6280 的 `2.4V~5.5V`，可以用于 `1.8V` 的低压系统；
2. 最大持续电流 `1.5A`；劣于 SY6280 的 `2A`；

### 引脚说明

| NO.  | NAME | I/O  | DESCRIPTION                                                  |
| ---- | ---- | ---- | ------------------------------------------------------------ |
| 1    | VIN  | I    | Device input                                                 |
| 2    | GND  | -    | Device ground                                                |
| 3    | CE   | I    | Active-low chip enable. Can be connected to VOUT for reverse current protection. Do not leave floating. |
| 4    | N/C  | -    | Not internally connected, can be tied to GND or left floating. |
| 5    | ST   | O    | Active-low open-drain output, pulled low when the chip is disabled. Hi-Z when the chip is enabled. Connect to GND if not required. |
| 6    | VOUT | O    | Device output                                                |

需要关注的有 2 个 IO：

1. `ST` 开漏输出，可输出当前芯片的工作状态；板载控制器可以通过该引脚检测该回路的电源工作状态；
2. `CE` 芯片片选输入引脚，这也是芯片最独特的地方，我们接下来详细看看。



## 电路原理应用

从芯片功能框图可以看出，这不是一个简单的高低电平片选引脚，而是一个输入到芯片内部比较器的反向输入端。

当 `CE` 引脚电压高于输入电压时，该器件被禁用，并且 `MOSFET` 关闭；

当 `CE` 引脚电压较低时，`MOSFET` 开启。

该功能因此衍生出了多种场景下的应用。

### 反向输入保护 RPP

![image-20260724151247278](https://picgocloud.com/m/e2bdcbfd-ecf1-4c3a-a7df-65eef0574b6a.png)

当应用于常规单向控制时，通过将 `CE` 接地，可以作为反接保护器件。

例如电池正负极接反时，`CE` 脚电平高于 `VIN`，则 `MOS` 管关闭，后端负载不受影响。

### 常开反向电流屏蔽 RCB

![image-20260724151954983](https://picgocloud.com/m/5956beff-2515-466b-970f-39bcecc2a7b5.png)

在常规的持续供电应用中，将 `CE` 脚连接至 `VOUT`，可以作为反向冲击保护器件。

正常情况下，`CE`、`VOUT` 和 `VIN` 电平一致，正常开启 `MOS` 管；

当 `VOUT` 受到外部强制异常冲击，导致电压升高，此时 `CE` 升高并大于 `VIN`，则主动关闭 `MOS` 管；

异常恢复后，`MOS` 管自动再打开。

### 双电源输入选通

![image-20260724155545414](https://picgocloud.com/m/47470ee5-bd1f-46b4-8540-4412b0496271.png)

如上就是该芯片更具价值的使用场景。若同时具有 2 个输入电源，不希望 2 处电源混合，例如 `VIN1` 来自 `DC-DC` 转换，`VIN2` 来自 USB 电源。

![image-20260724160044231](https://picgocloud.com/m/8dd2e89d-e50a-4810-abb8-e9b1c157298b.png)

此时通过 `CE` 引脚的交叉连接，`VIN` 更高的电源会失能对方的 `CE` 控制；

期间如果电压等级发生变化，`VOUT` 输出也能迅速调整为更高的电源电压输出，不会导致输出电源掉电。

这个方案也对应到了实际的一个项目应用，只是略微改变了 `CE` 的接法，如下图：

![image-20260724160729244](https://picgocloud.com/m/c4046881-825b-41a7-a729-2753609db361.png)

这种解法其实和上述类似，区别在于将 `CE` 都接到了统一输出端，也能够保护来自输出端的反向冲击。实际多个项目测试工作正常。

当然，上述电路还不够规范，`ST` 引脚按规格书要求，在不使用时最好接地。
